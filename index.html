FLVFixTool

The Ultimate Tool for Repairing and Editing FLV Files.

Easily inspect, repair corrupted timestamps, and edit metadata of your Flash Video (FLV) files. A native macOS application built for speed and simplicity.

Download for macOS

Only supported on MacOS 14.0 Sonoma or later.

Key Features

Inspect & Analyze

Dive deep into the structure of your FLV files. View headers, metadata, and individual tags in a clear, organized table.

Repair Corrupted Files

Automatically fix common FLV issues, such as incorrect durations and broken timestamps, making your files playable again.

Edit Metadata

Modify the `onMetaData` script tag with ease. Change dimensions, frame rates, and other key properties, then save your changes to a new file.
